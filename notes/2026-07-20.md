## 오늘의 개념: 벡터 임베딩(Vector Embedding)
벡터 임베딩이란 단어, 문장, 이미지 같은 데이터를 컴퓨터가 이해할 수 있는 숫자들의 목록(벡터)으로 바꾸는 방법입니다. 예를 들어 '고양이'와 '강아지'는 둘 다 동물이라는 의미적 유사성 때문에 숫자로 표현했을 때 서로 가까운 위치에 놓이게 됩니다. 반대로 '고양이'와 '자동차'는 의미가 많이 다르기 때문에 벡터 공간에서 멀리 떨어지게 됩니다. 이렇게 의미를 좌표처럼 표현하면, 컴퓨터가 텍스트나 이미지의 '비슷함' 또는 '관련성'을 계산할 수 있게 됩니다. 이 기술은 검색 엔진, 추천 시스템, 챗봇 등 다양한 AI 서비스에서 데이터를 이해하고 비교하는 핵심 기반 기술로 사용됩니다.

## 오늘의 용어: 벡터 임베딩 (Vector Embedding)
벡터 임베딩이란 단어, 문장, 이미지 같은 데이터를 컴퓨터가 이해할 수 있도록 숫자로 이루어진 목록(벡터)으로 변환하는 방법이에요. 예를 들어 '고양이'와 '강아지'라는 단어는 의미가 비슷하기 때문에 변환된 숫자 목록도 서로 가까운 위치에 놓이게 됩니다. 이렇게 하면 컴퓨터가 데이터 간의 유사성이나 관계를 계산하고 비교할 수 있게 됩니다.</definition>

## 예제 코드
```python
import numpy as np

# 각 단어를 3차원 벡터(임베딩)로 표현했다고 가정
# 실제로는 학습된 모델이 이런 벡터를 자동으로 생성함
word_vectors = {
    "고양이": np.array([0.9, 0.1, 0.2]),
    "강아지": np.array([0.8, 0.2, 0.1]),
    "자동차": np.array([0.1, 0.9, 0.8]),
}

# 코사인 유사도를 계산하는 함수 정의
def cosine_similarity(vec_a, vec_b):
    # 두 벡터의 내적(dot product) 계산
    dot_product = np.dot(vec_a, vec_b)
    # 각 벡터의 크기(노름, norm) 계산
    norm_a = np.linalg.norm(vec_a)
    norm_b = np.linalg.norm(vec_b)
    # 내적을 크기의 곱으로 나누어 유사도 반환 (-1 ~ 1 사이 값)
    return dot_product / (norm_a * norm_b)

# 고양이 vs 강아지: 의미상 비슷하므로 유사도가 높게 나올 것으로 기대
sim_cat_dog = cosine_similarity(word_vectors["고양이"], word_vectors["강아지"])

# 고양이 vs 자동차: 의미상 다르므로 유사도가 낮게 나올 것으로 기대
sim_cat_car = cosine_similarity(word_vectors["고양이"], word_vectors["자동차"])

# 결과 출력
print(f"고양이-강아지 유사도: {sim_cat_dog:.3f}")
print(f"고양이-자동차 유사도: {sim_cat_car:.3f}")

```
두 단어(또는 문장)를 간단한 숫자 벡터(임베딩)로 표현하고, 코사인 유사도를 계산해 두 벡터가 얼마나 비슷한 의미를 갖는지 확인하는 예제입니다. 실제로는 Word2Vec, BERT 같은 모델이 임베딩을 만들지만, 여기서는 개념 이해를 위해 임의의 벡터 값을 사용합니다. 유사도 값이 1에 가까울수록 두 벡터(단어)가 비슷하다는 의미입니다.

## 실무 팁
벡터 임베딩을 코사인 유사도로 비교할 계획이라면, 저장 전에 반드시 L2 정규화(normalize)를 해두세요. 그러면 벡터DB에서 내적(dot product) 연산만으로 코사인 유사도와 동일한 결과를 얻을 수 있어 검색 속도가 훨씬 빨라지고, FAISS 같은 라이브러리에서 IndexFlatIP를 그대로 사용할 수 있습니다.

Q. 벡터 임베딩(Vector Embedding)이란 무엇이며, 텍스트나 이미지 같은 데이터를 숫자로 이루어진 벡터로 변환할 때 어떤 원리로 '의미가 비슷한 것들은 벡터 공간에서 서로 가깝게' 배치되는지 설명해보세요.
A. 벡터 임베딩은 텍스트, 이미지 등의 데이터를 컴퓨터가 계산할 수 있도록 고차원 공간상의 숫자 벡터로 표현하는 방법입니다. 이때 모델은 데이터를 대량으로 학습하면서 의미적으로 비슷하거나 같은 맥락에서 자주 함께 등장하는 데이터들을 벡터 공간에서 서로 가까운 위치에 놓이도록 학습합니다. 예를 들어 '고양이'와 '강아지'는 벡터 공간에서 가까운 거리에 위치하지만, '고양이'와 '자동차'는 상대적으로 먼 거리에 위치하게 되며, 이 거리(유사도)를 코사인 유사도 등으로 계산해 데이터 간 의미적 관련성을 비교할 수 있습니다.
