## 오늘의 개념: 머신러닝이란 무엇인가
머신러닝은 컴퓨터가 사람이 일일이 규칙을 정해주지 않아도, 많은 예시(데이터)를 보고 스스로 패턴을 찾아내서 판단하는 능력을 갖게 하는 방법입니다. 예를 들어 아이에게 강아지 사진 100장과 고양이 사진 100장을 보여주면서 "이건 강아지야", "이건 고양이야"라고 계속 알려주면, 나중에 아이는 처음 보는 사진도 강아지인지 고양이인지 구별할 수 있게 됩니다. 컴퓨터도 이와 비슷하게 수많은 데이터를 보면서 "이런 특징이 있으면 강아지, 저런 특징이 있으면 고양이"라는 규칙을 스스로 만들어냅니다. 이렇게 사람이 직접 "귀 모양이 이러면 강아지"라고 하나하나 프로그래밍하지 않아도, 데이터를 통해 컴퓨터가 스스로 배우는 것이 머신러닝의 핵심입니다. 요즘 우리가 쓰는 유튜브 추천, 스팸 메일 필터링, 얼굴 인식 등이 모두 이 머신러닝 기술을 활용한 예시입니다.

## 오늘의 용어: 머신러닝 (Machine Learning)
머신러닝이란 컴퓨터에게 규칙을 하나하나 가르쳐주는 대신, 많은 예시(데이터)를 보여주고 컴퓨터가 스스로 패턴을 찾아내게 만드는 방법이에요. 예를 들어 아이에게 강아지 사진을 100장 보여주면서 "이게 강아지야"라고 알려주면, 나중에 아이가 처음 보는 강아지 사진도 스스로 알아보게 되는 것과 비슷해요. 즉, 사람이 직접 모든 규칙을 프로그래밍하지 않아도 컴퓨터가 경험(데이터)을 통해 점점 더 똑똑해지는 기술을 말합니다.</definition>

## 예제 코드
```python
> 
# 1) 머신러닝에 필요한 도구(라이브러리)를 가져옵니다.
# LinearRegression은 "숫자들 사이의 규칙(직선 관계)"을 찾아주는 도구입니다.
from sklearn.linear_model import LinearRegression

# 2) 문제(입력값)를 준비합니다. 여기서는 1, 2, 3, 4라는 숫자들입니다.
# 파이썬에서 [[ ]] 모양은 "표(테이블)"처럼 숫자를 정리해서 담는 방식입니다.
X = [[1], [2], [3], [4]]

# 3) 정답(출력값)을 준비합니다. 위 숫자들을 2배 한 값입니다.
# 즉, 컴퓨터에게 "1은 2가 되고, 2는 4가 되고..."라는 예시를 보여주는 것입니다.
y = [2, 4, 6, 8]

# 4) 머신러닝 모델(규칙을 찾아낼 상자)을 하나 만듭니다.
model = LinearRegression()

# 5) 모델에게 문제와 정답을 보여주며 규칙을 스스로 찾아내게 합니다.
# 이 과정을 "학습(training)"이라고 부릅니다.
model.fit(X, y)

# 6) 이제 학습한 모델에게 한 번도 보여주지 않은 숫자 5를 물어봅니다.
# 컴퓨터가 "5는 아마 10일 것 같아!"라고 스스로 예측하는 부분입니다.
prediction = model.predict([[5]])

# 7) 컴퓨터가 예측한 결과를 화면에 출력합니다.
print("5의 예측값:", prediction)

```
머신러닝이란 아주 쉽게 말하면, 컴퓨터에게 정답이 있는 여러 예시(문제와 답)를 잔뜩 보여주고 "이런 규칙이 있는 것 같아"라고 스스로 알아내게 만드는 것입니다. 마치 아이에게 여러 동물 사진과 이름을 계속 보여주면, 나중에 처음 보는 동물 사진을 보고도 "이건 강아지 같아!"라고 맞추는 것과 비슷합니다. 이 코드는 '숫자를 2배로 만드는 규칙'을 컴퓨터가 예시만 보고 스스로 알아내는 아주 간단한 머신러닝 예제입니다.

## 실무 팁
머신러닝(기계학습)이란 컴퓨터에게 규칙을 하나하나 직접 알려주는 대신, 아주 많은 예시(데이터)를 보여주고 스스로 패턴을 찾아내게 만드는 방법입니다. 예를 들어 아이에게 "이건 사과야, 이건 바나나야"라고 여러 번 과일 사진을 보여주면 나중에 처음 보는 과일도 스스로 구분하게 되는 것과 같은 원리입니다. 실제로 감을 잡아보고 싶다면, 코딩 지식이 전혀 없어도 구글에서 제공하는 무료 사이트 'Teachable Machine'(teachablemachine.withgoogle.com)에 들어가서 웹캠으로 자신의 손 모양(가위/바위/보)을 각각 20~30장씩 찍어 컴퓨터에게 학습시켜 보세요. 학습이 끝나면 컴퓨터가 실시간으로 여러분의 손 모양을 맞히는 걸 직접 볼 수 있는데, 이 과정 전체가 바로 '머신러닝'이 작동하는 원리를 그대로 체험하는 것입니다.
