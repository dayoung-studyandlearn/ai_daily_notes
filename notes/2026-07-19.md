## 오늘의 개념: Tool Use(함수 호출)
AI 언어모델은 원래 텍스트로 대화만 할 수 있지만, 실제로는 계산기, 날씨 조회, 데이터베이스 검색 같은 외부 기능이 필요한 경우가 많습니다. 'Tool Use(함수 호출)'는 AI가 이런 외부 도구나 함수를 스스로 판단해서 호출하고, 그 결과를 받아서 답변에 활용하는 기능을 말합니다. 예를 들어 "오늘 서울 날씨 알려줘"라고 물으면, AI는 직접 날씨를 알지 못해도 날씨 API라는 도구를 호출해서 실시간 정보를 가져온 뒤 사용자에게 답해줍니다. 이렇게 하면 AI가 단순히 학습한 지식에만 의존하지 않고, 실시간 정보 검색이나 정확한 계산처럼 스스로 하기 어려운 일을 외부 도구의 힘을 빌려 처리할 수 있게 됩니다. 즉, Tool Use는 AI를 더 똑똑하고 실용적으로 만들어주는 핵심 기술이라고 볼 수 있습니다.

## 오늘의 용어: Tool Use (함수 호출, Function Calling)
AI 모델이 스스로 답을 만들기 어려운 작업(예: 실시간 날씨 조회, 계산, 데이터베이스 검색)이 필요할 때, 미리 정의된 외부 함수나 도구를 호출해서 필요한 정보를 얻어오는 기능입니다. 즉, AI가 "이 작업은 내가 직접 못하니 이 도구를 써야겠다"고 판단하고 알맞은 함수를 실행해 결과를 답변에 활용하는 방식입니다. 이를 통해 AI는 단순 텍스트 생성뿐 아니라 실제 작업 수행 능력까지 갖추게 됩니다.

## 예제 코드
```python
# 도구(함수) 정의: 실제로는 API 호출, DB 조회 등이 될 수 있음
def get_weather(city):
    # 예시를 위해 고정된 날씨 정보를 반환
    return f"{city}의 날씨는 맑음, 22도입니다."

def add_numbers(a, b):
    # 간단한 덧셈 도구
    return a + b

# AI가 사용할 수 있는 도구 목록 (이름: 함수)
tools = {
    "get_weather": get_weather,
    "add_numbers": add_numbers
}

# 사용자의 요청을 분석해 어떤 도구를 호출할지 결정하는 간단한 함수
def choose_tool(user_input):
    if "날씨" in user_input:
        return "get_weather", {"city": "서울"}
    elif "더하기" in user_input:
        return "add_numbers", {"a": 3, "b": 5}
    return None, None

# 사용자 입력 예시
user_input = "서울 날씨 알려줘"

# 어떤 도구를 쓸지 결정
tool_name, args = choose_tool(user_input)

# 결정된 도구가 있으면 실행하고 결과 출력
if tool_name:
    result = tools[tool_name](**args)
    print("도구 실행 결과:", result)
else:
    print("적절한 도구를 찾지 못했습니다.")

```
AI 모델이 사용자의 질문에 답하기 위해 외부 '함수(도구)'를 호출하는 개념을 보여주는 예제입니다. 실제 LLM 대신 간단한 규칙으로 어떤 도구를 쓸지 결정하고, 해당 함수를 실행해 결과를 반환합니다. Tool Use의 기본 흐름(요청 -> 도구 선택 -> 도구 실행 -> 결과 반환)을 이해하는 데 도움이 됩니다.

## 실무 팁
LLM에 함수를 등록할 때는 파라미터 설명(description)을 최대한 구체적으로 작성하라. 예를 들어 "date: 날짜" 대신 "date: 조회할 날짜, 'YYYY-MM-DD' 형식, 예: '2024-05-01'"처럼 형식과 예시를 명시하면 모델이 잘못된 형식으로 함수를 호출하는 오류를 크게 줄일 수 있다. 또한 파라미터 개수는 5개 이하로 제한하고, 필수/선택 여부를 required 필드로 명확히 구분해야 호출 정확도가 높아진다.

## 오늘의 퀴즈
Q. AI 모델(LLM)이 실시간 날씨 정보나 계산기 같은 정확한 계산 결과를 사용자에게 답변해야 할 때, 왜 'Tool Use(함수 호출)' 기능이 필요한지 그 이유를 설명해보세요.
A. LLM은 학습 시점까지의 데이터를 기반으로 텍스트를 생성하는 방식이라 실시간 정보(현재 날씨, 최신 뉴스 등)를 알지 못하고, 복잡한 수치 계산도 정확하지 않을 수 있습니다. Tool Use를 사용하면 AI가 스스로 외부 API, 데이터베이스, 계산기 같은 실제 프로그램(함수)을 호출해서 정확하고 최신인 정보를 가져온 뒤, 그 결과를 바탕으로 답변을 생성할 수 있습니다. 즉, AI의 한계(실시간성 부족, 정확한 연산 불가)를 외부 도구와의 연동으로 보완하는 것이 핵심 개념입니다.
