# 실습 공간

`notes/`에서 배운 내용을 직접 코드로 만져보는 곳입니다.

`notes/`는 자동 생성되는 결과물이라 git에서 제외되어 있지만, 이 폴더는 직접 쓴 코드라서 git으로 추적합니다 —
나중에 커밋 기록을 보면 그날그날 무엇을 연습했는지 되짚어볼 수 있습니다.

## 쓰는 법

1. 그날 노트 날짜와 같은 이름으로 폴더를 만듭니다 (예: `2026-07-18/`).
2. 그 안에 원하는 이름으로 `.py` 파일을 만들어 자유롭게 연습합니다.
3. 정답은 없습니다 — 노트에서 본 개념을 직접 코드로 재현해보거나, 예제 코드를 변형해보는 용도입니다.

## 실행 방법

프로젝트 루트에서 가상환경을 켠 뒤 실행합니다.

```bash
.venv/Scripts/python.exe practice/2026-07-18/practice.py
```

**한글이 깨져 보인다면**: Windows 콘솔의 기본 문자 인코딩(cp949) 때문입니다. 코드는 정상 실행된 것이니, 아래처럼
`PYTHONIOENCODING` 환경 변수를 UTF-8로 지정하고 실행하면 해결됩니다.

```powershell
$env:PYTHONIOENCODING = "utf-8"
.venv/Scripts/python.exe practice/2026-07-18/practice.py
```
